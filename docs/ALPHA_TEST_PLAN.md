## 🐛 Bug分类

### 严重程度
- **Critical** (P0): 应用崩溃、数据丢失、安全漏洞
- **High** (P1): 核心功能无法使用、严重UI问题
- **Medium** (P2): 次要功能bug、性能问题
- **Low** (P3): UI小瑕疵、文案错误
