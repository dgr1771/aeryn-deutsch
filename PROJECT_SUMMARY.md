## 🏗️ 核心架构

### 1. 语法引擎系统

#### 已实现文件
- `lib/core/grammar_engine.dart` - 名词性别识别
- `lib/core/grammar_exercise_engine.dart` - 语法操练引擎（3000+题）

#### 覆盖语法点
```
A1-A2: 基础语法
├─ 四格（Kasus）
├─ 动词变位
├─ 形容词词尾
└─ 基础句型

B1-B2: 中级语法
├─ 从句（Nebensätze）
├─ 被动语态
├─ 虚拟式基础
├─ 介词搭配
└─ 句法结构

C1-C2: 高级语法
├─ 第二虚拟式高级
├─ 复杂句式嵌套
├─ 功能动词结构
├─ 学术德语
└─ 修辞手法
```
